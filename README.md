# avalanchesass_component_table
Table classes.

## Install
```
npm install avalanchesass_component_table --save
```

## Basic usage
```css
/* Import the main file. */
@import 'avalanchesass_component_table';

/* Import just the mixin file. */
@import 'avalanchesass_component_table/mixin';
```
